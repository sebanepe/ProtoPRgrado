# Plan de pruebas

Objetivo: Verificar componentes críticos del backend de detección de fraude.

- Unit tests: funciones de preprocesamiento, métricas, scoring.
- Integration tests: generación de alertas, registro/consulta de usuarios, importación de datasets.
- Regression: comparativa de métricas entre modelos guardados.
- Usabilidad: endpoints principales y flujo de configuración del modelo.
