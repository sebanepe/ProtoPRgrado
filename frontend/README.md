# Frontend - Fraud Detection System (Vite + React)

Run locally:

```bash
cd frontend
npm install
npm run dev
```

The frontend expects the backend API at `http://127.0.0.1:8000` by default. You can override with env var `VITE_API_BASE_URL`.

Pages:
- Login (simulated)
- Dashboard
- Import Data (multipart file upload)
- Preprocessing (button to trigger preprocessing)
- Models (list model metrics)
- Alerts (table of alerts)
- Users
- Settings

This is a minimal scaffold with reusable components.
